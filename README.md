# Análise de Atividade Física entre Universitários da UFSCar

## 📌 Sobre o Projeto
Este projeto analisa os hábitos de prática de atividade física entre estudantes da UFSCar (campus São Carlos), identificando proporção de praticantes, modalidades mais comuns, frequência de prática e principais motivos/barreiras.

## 📊 Principais Descobertas
- **57,1%** dos estudantes praticam atividade física regularmente
- **Musculação (62,5%)** e **Esportes Coletivos (53,1%)** são as modalidades mais praticadas
- **Falta de tempo (45,8%)** é a principal barreira para não praticantes
- Correlação fraca entre dias e horas de prática (r=0,34, p=0,06)

## 🛠️ Tecnologias Utilizadas
- Python 3.10+
- Pandas (análise de dados)
- NumPy (operações matemáticas)
- Matplotlib e Seaborn (visualizações)
- SciPy (estatística)
